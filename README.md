# Lottery Results Mobile Application

A Lottery Results Mobile Application built using Flutter, designed to provide users with real-time updates on lottery results. 


## Features

- Real-time lottery results updates
- User-friendly interface for easy navigation
- Support for multiple lottery games
- Notifications for winning numbers and jackpot updates
- Historical data and statistics for lottery games
- Secure and reliable data fetching from lottery APIs
- Cross-platform compatibility for both iOS and Android devices
- Dark mode support
- Widget support for quick access to lottery results on the home screen
- Localization support for multiple languages
